# E-commerce Sales Analytics SQL-Project

## Project Overview

This project analyzes sales, customers, inventory, marketing campaigns, and web analytics data using SQL, MySQL, Python, and Power BI.

## Objectives

- Perform data validation
- Conduct exploratory data analysis
- Analyze sales performance
- Analyze customer behavior
- Evaluate marketing effectiveness
- Monitor inventory
- Build executive dashboards

## Tools Used

- SQL
- MySQL
- getPass
- Python
- Pandas
- SQLAlchemy
- Power BI

## Dataset

The project uses an enterprise-style e-commerce dataset containing:

- Orders
- Customers
- Products
- Inventory
- Sales Rep
- Marketing Campaigns
- Customer Acquisition
- Web Analytics
